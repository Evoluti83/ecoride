# EcoRide

Projet réalisé dans le cadre du titre professionnel Développeur Web et Web Mobile.

## Description

EcoRide est une plateforme de covoiturage écologique permettant aux utilisateurs de partager leurs trajets et réduire leur impact environnemental.

## Technologies

- PHP
- MySQL
- HTML / CSS
- JavaScript
- Git / GitHub

## Structure du projet

assets : fichiers CSS et JS  
config : configuration de l'application  
controllers : logique métier  
models : interaction avec la base de données  
views : affichage HTML  
public : point d'entrée de l'application  
sql : scripts de base de données
